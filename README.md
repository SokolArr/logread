# Log reader (logread)
## About
- [x] 📝 convenient reading of logs
- [x] 🌎 opensource
- [x] 🐧 works on UNIX
- [ ] 🟰 regexp pattern *(beta)* to read log marks *(date, tag)*

## Quick start
1. make venv environment
2. install pyqt5
``` bash
    pip install pyqt5
```
- to build app
``` bash
    pyinstaller --onefile -w main.py
```
### Requirements
- pyqt5
